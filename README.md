## Design

- Egyetlen, önálló HTML fájl (Google Fonts CDN-en kívül nincs külső függőség)
- Színvilág: sötét tengerész/szürke (#1a2332) + narancs kiemelés (#E8530A)
- Betűtípusok: Inter (szöveg) + Playfair Display (cím)
- Teljes viewport magasságú diák (100vh)
- Nyomtatáshoz optimalizált: minden dia külön oldalon (@media print)
- Reszponzív: mobilon is használható

## CityOne adatok

- **3 park:** BudafokOne (15 000 m²), ÚjbudaOne (11 000 m²), KőbányaOne (District X)
- **~45 000 m²** összes terület
- **30+ bérlő**, **600+ dolgozó**
- **~10%** Budapest urban logistics piac
- **22%+** prémium szegmens részesedés
- **www.cityone.hu**
